# Frontend Performance Checklist

## Input

- Performance baseline measured
- Core Web Vitals analyzed
- Bundle size analyzed
- Network requests profiled
- Rendering performance measured

## Process

- Rendering optimizations identified
- Bundle size reductions planned
- Loading performance improvements defined
- Runtime performance optimizations selected
- Caching strategies designed

## Output

- Performance audit report complete
- Optimization recommendations documented
- Implementation plan created
- Performance budgets established
- Monitoring setup configured

## Quality

- Core Web Vitals meet targets (LCP < 2.5s, FID < 100ms, CLS < 0.1)
- Bundle size within budget
- No render-blocking resources
- Images optimized and lazy-loaded
- Critical rendering path optimized
- Performance monitoring active
- Automated performance testing in place
