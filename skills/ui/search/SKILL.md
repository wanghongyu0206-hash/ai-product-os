# Search Skill

## Goal

Design fast, intuitive, and scalable search experiences.

## Workflow

### Step 1

Identify search scope.

Examples:

- Global Search
- Page Search
- Knowledge Search
- Conversation Search

### Step 2

Determine searchable fields.

Examples:

- Name
- ID
- Email
- Status
- Tags
- Content

### Step 3

Design search interaction.

Include:

- Input Box
- Search Button (optional)
- Real-time Suggestions
- Recent Searches
- Clear Button

### Step 4

Design search results.

Display:

- Result Count
- Highlight Matched Keywords
- Sorting Options
- Pagination

### Step 5

Handle search states.

States:

- Initial
- Loading
- No Results
- Error

### Step 6

Generate search specification.
