# Interaction Skill

## Goal

Design predictable and user-friendly interactions for every page.

## Workflow

### Step 1

Identify user actions.

Examples:

- Click
- Input
- Upload
- Delete
- Search
- Save

### Step 2

Define system feedback.

Examples:

- Loading
- Success
- Failure
- Warning

### Step 3

Define confirmation actions.

Examples:

Delete

Archive

Reset

Logout

### Step 4

Define page states.

Examples:

Loading

Empty

No Permission

Error

Offline

### Step 5

Optimize efficiency.

Rules:

- Minimize unnecessary clicks.
- Keep feedback immediate.
- Prevent accidental operations.

### Step 6

Output interaction specification.
