## Design Specification Knowledge Base

A comprehensive collection of design system specifications is available in `references/design-system/awesome-design-md/`.

### How to Use

When the user asks for a specific UI style (e.g., "Apple style", "Vercel style", "Stripe style"), follow these steps:

1. **Navigate** to `references/design-system/awesome-design-md/design-md/<brand-name>/` and read the `DESIGN.md` file.
2. **Extract** the design tokens including colors, typography, spacing, border radius, shadows, and component patterns.
3. **Apply** the design tokens in your UI generation — whether in React code, Figma designs, or wireframes.
4. **Explain** which design system you used and your key design decisions.
