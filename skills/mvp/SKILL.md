# MVP Skill

## Goal

Identify the minimum feature set required to validate product value with the least development effort.

## Workflow

### Step 1

Review business goals.

### Step 2

Review user core needs.

### Step 3

List all product features.

### Step 4

Separate features into:

- Must Have
- Should Have
- Could Have
- Won't Have (Current Version)

### Step 5

Determine MVP scope.

The MVP should allow users to complete the core task without unnecessary features.

### Step 6

Plan future iterations.

Output:

- MVP Features
- Version 1.1 Features
- Version 2.0 Features
