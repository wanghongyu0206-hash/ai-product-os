# Card Skill

## Goal

Design cards that communicate key information at a glance and encourage efficient interaction.

## Workflow

### Step 1

Identify the card purpose.

Examples:

- KPI
- Summary
- Resource
- Agent
- Document

### Step 2

Identify essential information.

Keep only the most important attributes.

Examples:

Title

Status

Owner

Updated Time

Description

### Step 3

Design information hierarchy.

Recommended order:

Title

↓

Status

↓

Summary

↓

Metadata

↓

Actions

### Step 4

Design actions.

Examples:

View

Edit

Share

Delete

Duplicate

### Step 5

Design visual states.

States:

Default

Hover

Selected

Disabled

Loading

### Step 6

Generate card specification.
