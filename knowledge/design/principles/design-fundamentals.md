# Design Fundamentals

This document defines how AI understands UI design.

---

## 1. What is good design?

Good design is:

- Clear hierarchy
- Strong visual structure
- Reduced cognitive load
- Consistent spacing system
- Predictable interaction patterns

---

## 2. Design is not decoration

Design is NOT:

- adding colors
- adding shadows
- making things “beautiful”

Design IS:

- communication structure
- information hierarchy
- user decision flow

---

## 3. Layout principle

All UI should follow:

- Top → Down hierarchy
- Primary → Secondary → Tertiary information
- Strong grid system

---

## 4. Spacing system

Spacing is more important than color.

Recommended system:

- 4px / 8px / 16px / 24px / 32px / 48px

---

## 5. Typography rules

- One primary font
- Clear size hierarchy
- Avoid over-decoration

---

## 6. UI thinking model

Every UI should answer:

1. What is the user trying to do?
2. What is the primary action?
3. What is secondary information?

---

## 7. Rule for AI Agents

Before generating UI:

→ Read this document
→ Then select style
→ Then generate layout
