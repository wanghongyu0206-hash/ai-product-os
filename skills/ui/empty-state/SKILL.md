# Empty State Skill

## Goal

Transform empty pages into opportunities for user guidance and onboarding.

## Workflow

### Step 1

Identify why the page is empty.

Examples:

- First-time use
- No search results
- No permissions
- Data deleted
- Network issue

### Step 2

Choose the appropriate message.

Rules:

- Explain the situation.
- Avoid technical wording.
- Keep it concise.

### Step 3

Provide guidance.

Examples:

Create

Upload

Invite

Refresh

Retry

Clear Filters

### Step 4

Select visual elements.

Examples:

Illustration

Icon

Simple Graphic

Avoid excessive decoration.

### Step 5

Generate empty state specification.
