# Agent Fundamentals

This defines how AI agents behave.

---

## 1. What is an AI Agent?

An agent is NOT:

- a chatbot
- a prompt
- a tool

An agent IS:

a role that can reason + plan + execute.

---

## 2. Agent thinking flow

1. Understand request
2. Retrieve knowledge
3. Select strategy
4. Execute workflow
5. Produce output

---

## 3. Agent types

- Product Agent
- Designer Agent
- Engineer Agent
- Research Agent

---

## 4. Agent limitation

Agent MUST NOT:

- skip knowledge layer
- randomly choose style
- directly generate output

---

## 5. Agent principle

Think before act.

---

## 6. Multi-agent system

Complex tasks = collaboration:

Product → Design → Engineering

---

## 7. Rule

Agents ALWAYS use:

Knowledge → Style → Workflow → Tool
