# Product Fundamentals

This defines how AI understands product thinking.

---

## 1. What is a product?

A product is:

a system that solves a user problem.

NOT:

- a feature list
- a UI page
- a tool

---

## 2. Product thinking flow

Every product must answer:

1. Who is the user?
2. What problem do they have?
3. What is the simplest solution?
4. How does value appear?

---

## 3. PRD structure

A good PRD includes:

- Problem
- User
- Solution
- Scope
- Flow
- Success metrics

---

## 4. MVP principle

MVP = smallest useful version

NOT:

- full feature set
- perfect system

---

## 5. Product vs Feature

Feature = implementation

Product = outcome

---

## 6. Rule for AI Agents

Before generating PRD:

→ read this document
→ define user
→ define problem
→ simplify solution
