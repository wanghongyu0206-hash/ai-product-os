# Frontend Agent Checklist

## Input

- UI specification available from UI Designer Agent
- Frontend architecture available from Architect Agent
- Design tokens defined
- API contracts defined

## Process

- Project structure defined
- Component hierarchy designed
- Reusable components identified
- Code follows consistent patterns
- Design tokens applied correctly
- All interaction states implemented

## Output

- Frontend implementation complete
- Component structure documented
- API integration complete
- Loading, error, empty states handled
- Responsive behavior implemented

## Quality

- Maintainable code with clear separation of concerns
- Component reuse maximized
- Responsive on mobile, tablet, desktop
- Accessible (contrast, keyboard nav, ARIA)
- Performance optimized (lazy loading, memoization)
- Production ready
- QA Handoff Package complete
