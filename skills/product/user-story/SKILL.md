# User Story Skill

## Goal

Transform product requirements into clear and testable user stories.

## Workflow

### Step 1

Identify user roles.

Examples:

- Visitor
- Customer
- Administrator
- Operator

### Step 2

Identify user goals.

Examples:

- Submit an order
- Search knowledge
- Create a project

### Step 3

Write user stories.

Format:

As a ...

I want ...

So that ...

### Step 4

Define acceptance criteria.

Example:

Given ...

When ...

Then ...

### Step 5

Review completeness.

Ensure every core feature has at least one user story.
