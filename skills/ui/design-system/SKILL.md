# Design System Skill

## Goal

Create a consistent visual language that can be reused across all pages.

## Workflow

### Step 1

Define design principles.

Examples:

- Simplicity
- Consistency
- Clarity
- Accessibility

### Step 2

Standardize layout.

Include:

- Grid
- Page Width
- Content Width
- Section Spacing

### Step 3

Standardize typography.

Include:

- Heading Levels
- Body Text
- Caption
- Font Weight

### Step 4

Standardize components.

Examples:

Button

Input

Table

Card

Dialog

Tag

Badge

### Step 5

Standardize feedback.

States:

Success

Warning

Error

Loading

Empty

### Step 6

Generate design system specification.
