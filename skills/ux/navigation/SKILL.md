# Navigation Skill

## Goal

Create consistent and scalable navigation systems.

## Workflow

### Step 1

Identify navigation type.

Examples:

- Sidebar
- Top Navigation
- Bottom Navigation
- Tab Navigation

### Step 2

Define primary navigation.

Include only the main business modules.

Examples:

- Dashboard
- Customers
- Analytics
- Settings

### Step 3

Define secondary navigation.

Include related pages.

Examples:

Dashboard

- Overview
- Reports

Knowledge

- Documents
- Categories

### Step 4

Review naming.

Rules:

- Use concise names.
- Keep terminology consistent.
- Avoid duplicate meanings.

### Step 5

Review navigation depth.

Rules:

- Maximum depth: 3
- Avoid hidden pages.

### Step 6

Output final navigation structure.
