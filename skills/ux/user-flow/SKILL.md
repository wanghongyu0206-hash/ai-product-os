# User Flow Skill

## Goal

Design efficient user journeys that minimize friction and maximize task completion.

## Workflow

### Step 1

Identify the user role.

Examples:

- Visitor
- Customer
- Administrator

### Step 2

Define the user's goal.

Examples:

- Ask AI a question
- Upload knowledge
- Create a project

### Step 3

List all interaction steps.

Include:

- Entry
- Action
- System Response
- Exit

### Step 4

Identify decision points.

Examples:

- Login Required?
- Permission Check?
- Data Exists?

### Step 5

Optimize the flow.

Rules:

- Reduce unnecessary steps.
- Keep the primary task simple.
- Avoid repeated operations.

### Step 6

Output the complete user flow.
