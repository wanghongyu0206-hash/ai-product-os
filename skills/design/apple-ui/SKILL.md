# Apple UI Skill

## Goal

Generate premium Apple-style user interfaces.

---

## Workflow

Before generating any UI:

1. Read knowledge/principles.md

2. Read knowledge/colors.md

3. Read knowledge/spacing.md

4. Read components/

5. Choose suitable prompt template

6. Generate output

---

## Rules

Never invent random colors.

Always use the spacing system.

Always reuse component specifications.

Prefer whitespace over decoration.

Follow Apple Human Interface Guidelines.
