# Filter Skill

## Goal

Help users efficiently locate records by combining multiple filter conditions.

## Workflow

### Step 1

Identify filter dimensions.

Examples:

- Status
- Date
- Category
- Owner
- Priority

### Step 2

Select filter controls.

Examples:

- Select
- Multi Select
- Date Range
- Checkbox
- Switch

### Step 3

Design filter layout.

Order:

Frequently Used

↓

Business Attributes

↓

Advanced Filters

### Step 4

Display active filters.

Requirements:

- Visible filter tags
- One-click remove
- Clear all

### Step 5

Update result list.

Refresh automatically or after user confirmation.

### Step 6

Output filter specification.
