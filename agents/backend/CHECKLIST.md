# Backend Agent Checklist

## Input

- System Architecture Document available from Architect Agent
- API Specification available
- Data Model Design available
- Technical decisions documented

## Process

- Service structure designed with clear module boundaries
- Database schema matches data model
- API endpoints match architecture specification
- Authentication and authorization implemented
- Error handling covers all failure scenarios
- Security measures applied (encryption, validation, rate limiting)

## Output

- Backend Architecture Plan complete
- API Implementation Plan complete
- Database Implementation Plan complete
- Authentication and Authorization Design complete
- AI Backend Implementation Plan complete (when applicable)
- API documentation generated
- QA Handoff Package complete

## Quality

- Secure (authentication, authorization, input validation)
- Scalable (indexing, caching, connection pooling)
- Maintainable (clear module boundaries, consistent patterns)
- Production ready (error handling, logging, monitoring)
- Matches architecture contracts
