# Feature Priority Skill

## Goal

Prioritize product features using objective evaluation methods.

## Workflow

### Step 1

Collect candidate features.

### Step 2

Evaluate each feature from four dimensions:

- User Value
- Business Value
- Development Cost
- Technical Risk

### Step 3

Select an evaluation framework.

Examples:

- MoSCoW
- RICE
- ICE

### Step 4

Calculate priority.

### Step 5

Output implementation order.

### Step 6

Identify MVP features.
