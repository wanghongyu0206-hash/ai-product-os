# Competitive Analysis Skill

## Goal

Evaluate competitors and identify opportunities for product differentiation.

## Workflow

### Step 1

Identify direct competitors.

### Step 2

Identify indirect competitors.

### Step 3

Analyze each competitor from the following dimensions:

- Target Users
- Core Features
- Pricing
- User Experience
- Strengths
- Weaknesses

### Step 4

Compare products.

Focus on:

- Similarities
- Differences
- Missing Features

### Step 5

Identify opportunities.

Questions:

- Which user needs are not satisfied?
- Which features can become competitive advantages?

### Step 6

Output product positioning.
