# AI Service Checklist

## Input

- AI capabilities identified
- LLM providers selected
- Throughput and latency requirements defined
- Cost constraints established
- Compliance requirements clarified

## Process

- Service architecture designed
- LLM integration layer implemented
- Prompt engineering pipeline created
- AI workflow orchestration designed
- Cost management strategy planned
- Safety and compliance measures implemented

## Output

- AI service architecture diagram complete
- LLM integration layer documented
- Prompt library created and versioned
- Workflow orchestration designed
- Cost management strategy documented
- Safety plan implemented

## Quality

- LLM integration is provider-agnostic
- Error handling is robust (retries, fallbacks)
- Cost management is comprehensive
- Prompts are versioned and tested
- Content filtering is implemented
- Monitoring covers usage, latency, errors
- API is well-documented
- Compliance requirements are met
