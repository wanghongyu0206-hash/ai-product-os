# Wireframe Skill

## Goal

Create clear page structures that prioritize usability over visual appearance.

## Workflow

### Step 1

Identify the page purpose.

Examples:

- Dashboard
- List Page
- Detail Page
- Form Page
- Settings

### Step 2

Identify primary user task.

Examples:

- View data
- Search information
- Submit a form
- Configure settings

### Step 3

Define page regions.

Typical regions:

- Header
- Navigation
- Main Content
- Sidebar (optional)
- Footer (optional)

### Step 4

Arrange content by importance.

Priority:

1. Primary task
2. Supporting information
3. Secondary actions

### Step 5

Place UI components.

Examples:

- Table
- Form
- Card
- Chart
- Tabs
- Search
- Filter

### Step 6

Review layout.

Check visual hierarchy and task completion efficiency.
