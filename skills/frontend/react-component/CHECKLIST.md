# React Component Checklist

## Props

- Props interface defined
- Required props identified
- Default values set
- TypeScript types assigned

## Composition

- Single responsibility followed
- Composition pattern chosen
- Reusability maximized
- Children/slots supported

## State

- State management approach defined
- Derived state identified
- Side effects handled correctly
- Performance optimized

## Quality

- Accessibility implemented
- Tests written
- Documentation complete
- API is intuitive
