# Product Roadmap Skill

## Goal

Plan product evolution from MVP to long-term growth.

## Workflow

### Step 1

Review product vision.

### Step 2

Review MVP.

### Step 3

Collect remaining features.

### Step 4

Group features by business value.

### Step 5

Define release versions.

Examples:

Version 1.0

Version 1.1

Version 2.0

### Step 6

Define milestones.

Examples:

Internal Testing

Beta Release

Public Release

Enterprise Version

### Step 7

Generate roadmap summary.
