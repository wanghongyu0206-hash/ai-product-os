# Architect Agent Checklist

## Input

- Requirements complete from Product Manager Agent
- Business goals clear
- Technical constraints identified

## Process

- Architecture designed with clear module boundaries
- System boundaries defined
- Data model designed with entities and relationships
- API structure specified with data contracts
- Technical decisions documented with tradeoffs

## Output

- System Architecture Document complete
- Technical Decision Document complete
- Data Model Design complete
- API Design Specification complete
- AI Architecture Design complete (when applicable)
- Engineering Handoff Package ready

## Quality

- Scalable architecture that supports growth
- Maintainable with clear separation of concerns
- Implementation ready for Backend and Frontend Agents
- All decisions justified with reasoning
