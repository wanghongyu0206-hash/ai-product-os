# Settings Skill

## Goal

Organize system settings into logical categories for efficient management.

## Workflow

### Step 1

Identify configuration domains.

Examples:

- Account
- Team
- Security
- Permissions
- API
- Billing

### Step 2

Group related settings.

Recommended groups:

Account

Organization

Access Control

Integration

System Preferences

### Step 3

Design page structure.

Recommended layout:

Sidebar

↓

Category List

↓

Configuration Panel

↓

Save Actions

### Step 4

Design actions.

Examples:

Save

Reset

Generate API Key

Invite Member

Remove User

### Step 5

Design confirmation flow.

Dangerous actions:

Delete Workspace

Regenerate API Key

Remove Member

Require confirmation.

### Step 6

Generate settings specification.
