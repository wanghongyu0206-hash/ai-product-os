# Knowledge Layer

This is the core cognitive system of AI Product Studio.

It defines how AI understands:

- Design
- Product
- Engineering
- AI Systems

---

## Structure

- design → visual & UI thinking
- product → product thinking & logic
- engineering → implementation logic
- ai → intelligence & agent systems

---

## How it is used

Agents MUST reference Knowledge before:

- selecting styles
- generating UI
- writing PRD
- creating workflows

---

## Rule

Knowledge is NOT execution.

Knowledge is reasoning foundation.
