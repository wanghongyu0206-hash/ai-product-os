# Frontend Development Workflow


## Purpose

Deliver production-ready frontend applications from design specifications.


## Trigger

Frontend implementation requested.


## Agents


Architect

↓

Frontend

↓

QA


## Workflow


### Step 1: Frontend Architecture

Agent:

architect


Input:

- UI Specification


Output:

- Frontend Architecture Plan



### Step 2: Component Implementation

Agent:

frontend


Input:

- UI Specification
- Architecture Plan


Output:

- React Components
- Pages
- Styles



### Step 3: Performance Optimization

Agent:

frontend


Input:

- Frontend Application


Output:

- Performance Report



### Step 4: Quality Review

Agent:

qa


Input:

- Frontend Deliverables


Output:

- Frontend Review Report


## Completion Criteria

- Frontend implemented
- Performance validated
- Review completed
