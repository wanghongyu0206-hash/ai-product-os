# PRD Skill

## Overview

This skill helps AI generate structured and professional Product Requirement Documents (PRDs).

## When to Use

Use this skill for:

- New product planning
- Feature design
- Requirement analysis
- AI product design
- SaaS planning

## Output

The AI will generate:

- Product background
- User analysis
- User stories
- Functional requirements
- Metrics
- Event tracking
- Acceptance criteria
