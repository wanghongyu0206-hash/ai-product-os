# Table Skill

## Goal

Design highly efficient data tables for enterprise products.

## Workflow

### Step 1

Identify the business object.

Examples:

- Customer
- Conversation
- Document
- User
- Order

### Step 2

Determine the primary task.

Examples:

- Browse
- Search
- Edit
- Delete
- Export

### Step 3

Design the toolbar.

Typical components:

- Search
- Filter
- Create
- Batch Actions
- Export

### Step 4

Design table columns.

Priority:

Primary Identifier

↓

Status

↓

Important Attributes

↓

Time

↓

Actions

### Step 5

Design row actions.

Examples:

View

Edit

Delete

Duplicate

### Step 6

Design pagination.

Include:

- Total count
- Page size
- Page navigation

Output the final table specification.
