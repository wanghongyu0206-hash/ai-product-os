# Form Skill

## Goal

Design forms that minimize user effort while ensuring data quality.

## Workflow

### Step 1

Identify the business object.

Examples:

- User
- Document
- Knowledge Base
- Agent
- API Key

### Step 2

List all fields.

Separate fields into:

- Required
- Optional
- System Generated

### Step 3

Group related fields.

Typical groups:

Basic Information

Business Settings

Permissions

Advanced Settings

### Step 4

Select appropriate controls.

Examples:

Short Text

Textarea

Select

Multi Select

Radio

Checkbox

Date Picker

Upload

Switch

### Step 5

Define validation.

Examples:

Required

Format

Length

Duplicate Check

Permission Check

### Step 6

Design submit flow.

Save

↓

Validation

↓

Loading

↓

Success

↓

Return / Continue Editing

Output complete form specification.
