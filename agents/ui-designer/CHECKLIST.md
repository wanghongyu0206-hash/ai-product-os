# UI Designer Agent Checklist

## Input

- Information Architecture available
- User Flow available
- Wireframe Specification available
- Interaction Specification available

## Process

- Design direction selected and documented
- Design system tokens defined with exact values
- All required components identified
- Component states defined (default, hover, active, disabled, loading, error)
- Responsive behavior documented

## Output

- Visual Design Direction complete
- Design Token Specification complete
- Component Specification complete
- Page Layout Specification complete
- Figma Design Brief prepared
- Frontend Handoff Package complete

## Quality

- Consistent visual language across all pages
- All components use design system tokens
- Responsive behavior defined (mobile, tablet, desktop)
- Component reuse maximized
- No orphan components
- Developer-ready specification with exact values
- Handoff format compatible with Frontend Agent
