# Accessibility Skill

## Goal

Ensure products are usable for all users, including people with disabilities.

## Workflow

### Step 1

Review page structure.

Check:

- Heading hierarchy
- Navigation order
- Content grouping

### Step 2

Review keyboard accessibility.

Check:

- Tab order
- Focus visibility
- Keyboard shortcuts

### Step 3

Review readability.

Check:

- Font size
- Line spacing
- Content hierarchy

### Step 4

Review color usage.

Check:

- Text contrast
- Status colors
- Interactive elements

### Step 5

Review feedback.

Check:

- Error messages
- Success messages
- Loading indicators

### Step 6

Generate accessibility recommendations.
