# Apple UI Skill

## Overview

Apple UI Skill helps AI generate user interfaces inspired by Apple's Human Interface Guidelines (HIG).

This skill focuses on:

- Clean layouts
- Elegant typography
- Generous white space
- Premium visual design
- Modern interaction patterns
- Accessibility
- Responsive interfaces

---

## When to Use

Use this skill when designing:

- AI applications
- SaaS products
- Dashboards
- Landing pages
- Admin panels
- Mobile interfaces
- Chat applications
- Productivity tools

---

## What This Skill Provides

This skill guides the AI to generate:

- Layout structure
- Component hierarchy
- Apple-inspired visual style
- UX interaction suggestions
- Accessibility recommendations
- Production-ready React implementation

---

## Design Principles

This skill follows Apple's design philosophy:

- Simplicity
- Consistency
- Clarity
- Depth
- Large white space
- Natural motion
- High accessibility

---
