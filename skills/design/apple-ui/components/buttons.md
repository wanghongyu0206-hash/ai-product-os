# Apple Button Guidelines

## Primary Button

Purpose:

Main action on the page.

Style:

- Filled background
- Rounded corners (12px)
- Height: 44px
- Horizontal padding: 16px
- Bold text

---

## Secondary Button

Purpose:

Secondary actions.

Style:

- Outline
- White background
- Gray border

---

## Destructive Button

Purpose:

Delete or irreversible actions.

Style:

- Red text
- Red outline
- Confirmation required

---

## Disabled Button

Opacity: 40%

No hover effect.

No shadow.
