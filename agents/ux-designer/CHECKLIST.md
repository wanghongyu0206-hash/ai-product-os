# UX Designer Agent Checklist

## Input

- PRD.md received from Product Manager Agent
- User Stories available
- Feature Requirements defined
- User Personas identified

## Process

- User goals analyzed and documented
- Information Architecture created (depth ≤ 3)
- User Flow designed for all core tasks
- Interaction Specification defined
- Wireframe Specification created
- UX Review completed

## Output

- Information Architecture document complete
- User Flow diagrams complete
- Interaction Specification document complete
- Wireframe Specification document complete
- UX Review Report generated
- Handoff data formatted for UI Designer Agent

## Quality

- Navigation depth ≤ 3 levels
- Every page has one clear responsibility
- All core user flows complete
- Interaction states defined (loading, empty, error, success)
- Accessibility requirements met
- No unnecessary steps in user flows
- Developer-ready specifications
