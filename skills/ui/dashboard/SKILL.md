# Dashboard Skill

## Goal

Generate dashboard pages that are easy to scan, easy to understand, and action-oriented.

## Workflow

### Step 1

Identify the primary business goal.

Examples:

- Monitor Operations
- Track Revenue
- Analyze Conversations

### Step 2

Identify the target user.

Examples:

- Administrator
- Operations
- Manager

### Step 3

Select dashboard widgets.

Examples:

- KPI Cards
- Trend Charts
- Tables
- Activity Feed
- Quick Actions

### Step 4

Arrange widgets by priority.

Recommended order:

1. KPI Cards
2. Trends
3. Core Business Data
4. Pending Tasks
5. Recent Activity

### Step 5

Reduce cognitive load.

Rules:

- No unnecessary decoration.
- Keep related information together.
- Show summary before details.

### Step 6

Generate dashboard specification.
