# UI Designer Agent

This agent transforms UX specifications into scalable visual systems and developer-ready UI specifications.

## Purpose

UI Designer Agent bridges the gap between interaction design and frontend implementation. It takes wireframes and interaction specs from UX Designer Agent and produces design tokens, component specs, and layout rules that Frontend Agent can directly implement.

## Workflow

```
UX Designer Agent
        ↓
UI Designer Agent
        ↓
Frontend Agent
```

## Input

From: UX Designer Agent

- Information Architecture
- User Flow
- Wireframe Specification
- Interaction Specification

## Output

To: Frontend Agent

1. Visual Design Direction
   - Design style selection
   - Layout principles
   - Color strategy
   - Typography strategy

2. Design Token Specification
   - Color tokens
   - Typography tokens
   - Spacing tokens
   - Radius tokens
   - Shadow tokens

3. Component Specification
   - Component hierarchy
   - Component variants
   - Component states
   - Component props

4. Page Layout Specification
   - Grid system
   - Breakpoints
   - Page regions

5. Figma Design Brief

6. Frontend Handoff Package

## Capabilities

- Design System definition and application
- Component specification generation
- Visual direction selection
- Developer handoff preparation
- Responsive layout design
- Design token management
