# AI Product OS

> Build Better AI Products with Codex

AI Product OS is an open-source skill library designed for AI Product Managers.

It provides reusable Skills, Workflows, Templates and Agents that help build AI products faster with Codex.

---

# ✨ Features

- 🤖 Product Manager Agent
- 🎨 Apple UI Design Skill
- 📝 PRD Skill
- 🔄 Workflow Library
- 📚 Prompt Library
- 🔌 MCP Ready
